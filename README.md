# 研究生学术人格测验

一个娱乐向网页测试：通过 36 道研究生情境题，测出你的“学术角色”人格，并展示 MBTI 风格人物卡、匹配排行和生存建议。

在线体验：https://maoqiu.vip/grad-persona-test/

![研究生学术人格测验分享图](./grad-persona-share.png)

## 特色

- 36 道研究生情境题，按第一反应作答。
- 16 种学术人格角色，覆盖组会、论文、投稿、DDL 和摸鱼场景。
- 自动生成结果图，支持分享图片或长按保存。
- 移动端单屏答题体验，适合手机浏览器和微信内打开。

角色包括：学术沙僧、学术葫芦娃、学术袁华、学术爱德华、学术刘姥姥、学术八戒、学术吕布、学术孟获、学术灰姑娘、学术鲁滨逊、学术孙悟空、学术唐僧、学术林黛玉、学术孔乙己、学术曹操、学术哪吒。

## 使用

直接打开 `index.html` 即可本地游玩，也可以访问线上版本。
